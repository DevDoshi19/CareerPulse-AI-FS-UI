# Security Policy

## Supported Versions

Since this project is currently in active development, only the latest version is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of CareerForge AI seriously. If you discover a security vulnerability, please follow these steps:

1.  **Do NOT open a public issue.** This allows us to fix the vulnerability before it can be exploited by others.
2.  **Email the details** privately to: **devdoshi1927@gmail.com**
3.  Include as much information as possible:
    * Type of vulnerability (e.g., XSS, API Key leakage).
    * Steps to reproduce the issue.
    * Any relevant code snippets or screenshots.

### Our Response
* We will acknowledge your report within **48 hours**.
* We will provide an estimated timeline for the fix.
* Once fixed, we will notify you and release a patch.

Thank you for helping keep CareerForge AI safe!
